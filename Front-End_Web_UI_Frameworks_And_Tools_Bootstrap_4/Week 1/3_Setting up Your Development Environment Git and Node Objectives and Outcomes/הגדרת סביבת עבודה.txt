הגדרת סביבת הפיתוח שלך
דרישות תוכנה

* עורך טקסט לבחירתך: 
כל עורך טקסט שאתה כבר מכיר יכול לשמש לעריכת קבצי הפרויקט. 
אני אשתמש ב- Visual Studio Code (https://code.visualstudio.com/) כעורך המועדף בהתמחות זו. 
תוכל לשקול גם עורכים אחרים כגון Brackets (http://brackets.io/), טקסט נשגב (http://www.sublimetext.com/) או Atom (https://atom.io/).

* דפדפן לבחירתך: 
תוכל להשתמש בדפדפן המועדף עליך. 
אני אשתמש ב- Chrome כדפדפן בכל התרגילים. 
כל התרגילים והמטלות בקורס זה נבדקו באמצעות Chrome v. 46. 
שים לב שלא כל הדפדפנים עשויים לתמוך בכל תכונות HTML5 באותה מידה. 
אתה עלול להיתקל בבעיות בעת שימוש בדפדפנים אחרים. 
אני ממליץ לך בחום להשתמש בדפדפן העדכני ביותר של Chrome לתרגילים ולמטלות בקורס זה, כך שהבעיות יהיו ממוזערות.

* מעטפת שורת הפקודה: 
הכרות עם מעטפת שורת הפקודה תהיה חיונית לתרגילים. 
ב- Windows יהיה צורך בחלון cmd או במעטפת כוח עם הרשאות ניהול. 
ב- Mac או ב- Linux, ניתן להשתמש בחלון מסוף. אנא הכירו את הפקודה "sudo" במערכת ההפעלה X ו- Linux.

* הקבצים הנדרשים לתרגילים: 
אנו נספק קבצי התחלה נוספים לתרגילים בכל מקום שצריך. 
קישורים להורדת הקבצים יסופקו מוטמעים בהוראות התרגיל שאחרי כל סרטון תרגיל. 
אנא הורד את הקבצים המסופקים שם, אם יש כאלה, לפני תחילת התרגיל. הקישורים זמינים גם באמצעות המשאבים הנוספים של השיעור הספציפי.

* הערה: 
זכור לשמור את התיקיות ואת כל הקבצים שאתה יוצר בתרגילים. 
תרגילים נוספים יבנו על הקבצים שתיצור בתרגילים הקודמים. 
אין למחוק את הקבצים בתום התרגילים, אלא אם כן הורה אחרת. 
ייתכן שתרצה להגדיר את תיקיית האימון שלך כמאגר Git ולהעביר את הקבצים למאגר בסוף כל תרגיל.
